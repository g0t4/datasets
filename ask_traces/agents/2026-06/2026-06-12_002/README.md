Use a Markdown blockquote. Put a `>` at the start of the line (or at the start of each line if it’s multiline).
The quote can contain normal Markdown formatting, like **bold**, and you don’t need any extra escaping for the surrounding double‑quotes.

```markdown
> "denotes tables passed by **key**"
```

Result:

> "denotes tables passed by **key**"

If you need a nested quote, add another `>`:

```markdown
> > "inner quote"
```
